🔹 What is SQL?

SQL (Structured Query Language) is a standardized programming language used for managing and manipulating relational databases. Whether you're analyzing business trends, building web apps, or powering dashboards, SQL is a fundamental skill for working with data efficiently.

🔸 Core SQL Queries

SELECT name, age FROM students;
INSERT INTO students (name, age) VALUES ('Alice', 22);
UPDATE students SET age = 23 WHERE name = 'Alice';
DELETE FROM students WHERE age < 18;
SELECT * FROM students WHERE age > 20;
SELECT s.name, c.course FROM students s JOIN courses c ON s.id = c.student_id;
SELECT * FROM students WHERE name LIKE 'A%';
SELECT * FROM students WHERE age IN (20, 21, 22);
SELECT age, COUNT(*) FROM students GROUP BY age;
SELECT * FROM students ORDER BY age DESC;
SELECT * FROM students LIMIT 5;
