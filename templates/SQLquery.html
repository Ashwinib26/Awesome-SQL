SQL (Structured Query Language) is the language used to communicate with databases ‚Äî to insert, retrieve, update, and manage data efficiently.

üî∏ Core SQL Concepts

Used to retrieve data from one or more tables.
eg. SELECT name, age FROM students;

Adds new rows of data into a table.
eg. INSERT INTO students (name, age) VALUES ('Alice', 22);

Modifies existing records in a table.
eg. UPDATE students SET age = 23 WHERE name = 'Alice';

Removes rows from a table.
eg. DELETE FROM students WHERE age < 18;

Filters records based on specific conditions.
eg. SELECT * FROM students WHERE age > 20;

Combines rows from two or more tables based on a related column.
eg. SELECT s.name, c.course FROM students s JOIN courses c ON s.id = c.student_id;
