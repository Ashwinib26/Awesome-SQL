üîπ What is SQL?

SQL (Structured Query Language) is a standardized programming language used for managing and manipulating relational databases.
Whether you're analyzing business trends, building web apps, or powering dashboards, SQL is a fundamental skill for working with data efficiently.

üî∏ Core SQL Concepts

The SELECT statement fetches data from one or more tables.

SELECT name, age FROM students;

Use INSERT to add new records to a table.

INSERT INTO students (name, age) VALUES ('Alice', 22);

UPDATE changes existing values in one or more rows.

UPDATE students SET age = 23 WHERE name = 'Alice';

DELETE permanently removes rows that meet a condition.

DELETE FROM students WHERE age < 18;

The WHERE clause restricts results based on conditions.

SELECT * FROM students WHERE age > 20;

A JOIN merges rows from multiple tables based on relationships.

SELECT s.name, c.course FROM students s JOIN courses c ON s.id = c.student_id;

Used to search for a specified pattern in a column.
SELECT * FROM students WHERE name LIKE 'A%';

Filters records where a column value is in a specified list.
SELECT * FROM students WHERE age IN (20, 21, 22);

Groups rows that have the same values into summary rows.
SELECT age, COUNT(*) FROM students GROUP BY age;

Sorts result set in ascending or descending order.
SELECT * FROM students ORDER BY age DESC;

Limits the number of rows returned by a query.
SELECT * FROM students LIMIT 5;
